# AI Studio Landing Page Project

## 📌 Project Overview

This project is a structured landing page application designed to manage and display a dynamic catalog of cleaning services and related offerings. It is built to support easy content updates, scalability, and future integration of business tools such as CRM systems and marketing automation services.

The architecture is intentionally modular, allowing future enhancements without major refactoring.

---

## 🎯 Key Objectives

- Provide a clean and maintainable landing page structure
- Enable easy updates of visual and textual content
- Serve as a scalable foundation for a service catalog system
- Prepare infrastructure for CRM and business automation integration

---

## 🧩 Planned / Integrated Features

- Dynamic landing page content system
- Service catalog for cleaning and related services
- Scalable architecture for future modules
- CRM integration (HubSpot)
- Database-driven content management (Supabase)
- Production deployment via Vercel
- Version-controlled development workflow using GitHub

---

## 🛠 Tech Stack

- **Frontend:** React (AI Studio generated app structure)
- **Runtime:** Node.js
- **API Integration:** Gemini API
- **Database:** Supabase
- **CRM:** HubSpot (planned integration)
- **Deployment:** Vercel
- **Version Control:** GitHub
